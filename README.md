## ForSind: Seu Síndico no Bolso! 

**O que é o ForSind?**

O ForSind é uma aplicação web que facilita a vida dos síndicos, permitindo a busca rápida e eficiente de informações sobre condomínios. Com apenas alguns cliques, você encontra tudo o que precisa saber sobre um determinado condomínio.

**Como funciona?**

1. **Pesquisa intuitiva:** Digite o nome do condomínio ou parte do endereço na barra de pesquisa e pressione Enter.
2. **Resultados instantâneos:** A aplicação exibe uma lista com todos os condomínios que correspondem à sua pesquisa.
3. **Informações detalhadas:** Para cada condomínio, você encontra informações como nome, endereço completo e status de atividade.
